# Project: AI Demo - Card News Generator (ai-demo)

## Goal
공공기관 CMS 담당자를 위한 **'AI 기반 카드뉴스 자동 생성'** 프로토타입 구현.

## Core Features
- **Smart Summary**: 게시물 본문을 분석하여 핵심 요약문과 키워드 자동 추출.
- **Card News Generation**: 요약문을 바탕으로 '심플형' 디자인의 SNS 카드뉴스 이미지 생성 및 텍스트 레이아웃 배치.
- **UI/UX Optimization**: CMS 웹에디터 옆에 AI 사이드바를 배치하여 실시간 확인 및 원클릭 생성 지원.

## Success Criteria
- [ ] 웹에디터 입력 시 AI 사이드바에 요약문이 실시간(모의)으로 노출됨.
- [ ] [카드뉴스 생성] 버튼 클릭 시 요약 컨텐츠와 어울리는 이미지가 배경으로 포함된 카드 이미지가 생성됨.
- [ ] 디자인 시스템이 적용된 프리미엄 UI 유지.
