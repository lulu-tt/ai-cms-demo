# Requirements: AI Card News Generator

## 1. 개요
공공기관 담당자가 게시물을 등록할 때, 별도의 디자인 도구 없이 AI만으로 고품질 카드뉴스를 제작할 수 있도록 지원하는 기능 요구사항.

## 2. 기능적 요구사항
- **F1. 실시간 요약 (Real-time Summary)**:
    - 사용자가 에디터에 텍스트를 입력하면 AI가 핵심 3줄 요약을 화면 우측 사이드바에 표시한다.
- **F2. 이미지 자동 매칭 (Image Matching)**:
    - 요약문 키워드에 따라 배경 이미지를 추천/생성한다.
- **F3. 카드뉴스 레이아웃 (Layout Composition)**:
    - '심플형' 디자인(고급스러운 배경 + 중앙/하단 텍스트)을 적용하여 카드 이미지를 렌더링한다.
- **F4. 간단 편집 (Simple Edit)**:
    - 생성된 카드뉴스 위의 텍스트를 사용자가 직접 수정할 수 있어야 한다.

## 3. 비기능적 요구사항
- **디자인 가이드**: 공공기관의 신뢰성을 줄 수 있는 깔끔한 타이포그래피와 색상을 사용한다.
- **사용성**: 복잡한 설정 없이 [생성] 버튼 하나로 결과물이 나와야 한다.
